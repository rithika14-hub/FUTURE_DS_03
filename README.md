# Customer Conversion & Campaign Funnel Analysis

## Project Overview
This project focuses on analyzing customer conversion behavior and marketing campaign performance using a bank marketing dataset. The analysis was performed using Python and Microsoft Excel to identify conversion patterns, customer segments, and campaign effectiveness.

## Objectives
- Analyze customer conversion behavior
- Evaluate campaign performance
- Identify high-converting customer segments
- Create interactive dashboards
- Generate business insights and recommendations

## Tools & Technologies
- Python
- Google Colab
- Microsoft Excel
- Pivot Tables & Charts

## Project Workflow
1. Data Cleaning & Preprocessing
2. Exploratory Data Analysis (EDA)
3. Funnel & Conversion Analysis
4. Dashboard Creation
5. Insight Generation

## Dashboard Features
- Conversion Distribution Analysis
- Contact Method Analysis
- Job-wise Conversion Analysis
- Monthly Conversion Trends
- Balance-based Conversion Analysis
- Funnel Chart Visualization
- Interactive Slicers

## Key Insights
- Majority of customers did not convert
- Cellular contact showed better conversion rates
- Certain job categories contributed more conversions
- Campaign timing impacted conversion performance
- High-balance customers converted more frequently
